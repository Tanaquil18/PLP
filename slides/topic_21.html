Topic 21—Generalising induction (a bit)

Demirbaş & Rechnitzer

Two generalisations of induction

A generalisation

Theorem: Mathematical induction.

Let \(\ell \in \mathbb{Z}\) and \(S = \set{n \in \mathbb{Z}\st n \geq \ell}\text{.}\) Let \(P(n)\) be a statement for all \(n \in S\text{.}\) Then if

  • \(P(\ell)\) is true, and
  • \(P(k) \implies P(k+1)\) is true for all integer \(k \in S\)
then \(P(n)\) is true for all \(n \in S\text{.}\)

Proposition:

For every integer \(n \geq 5\text{,}\) \(2^n \geq n^2\)

Proof

Proof.

We prove the result by induction. Since \(2^5 = 32 \gt 25 = 5^2\text{,}\) the result holds when \(n=5\text{.}\) Now assume that \(k\geq 5\) and that \(2^k \geq k^2\text{.}\) Then

\begin{align*} 2^{k+1} \amp \geq 2k^2 = k^2+k^2 \\ \amp \geq k^2 + 5k \amp \text{since } k \geq 5\\ \amp = k^2 + 2k + 3k \\ \amp \geq k^2 + 2k +1 \amp \text{since } k \geq 5 \end{align*}
Thus the inductive step holds for \(k \geq 5\text{.}\)

The result follows for all integer \(n \geq 5\) by induction.

Another generalisation

Theorem: Strong mathematical induction.

Let \(\ell \in \mathbb{Z}\) and \(S = \set{n \in \mathbb{Z}\st n \geq \ell}\text{.}\) Let \(P(n)\) be a statement for all \(n \in S\text{.}\) Then if

  • \(P(\ell)\) is true, and
  • \(\left( P(\ell) \land P(\ell+1) \land P(\ell+2) \land \cdots \land P(k) \right) \implies P(k+1)\) is true for all integer \(k \in S\)
then \(P(n)\) is true for all \(n \in S\text{.}\)

Proposition:

Let \(\theta \in \mathbb{R}\) be fixed.

Let \(p_0 = 1, p_1 = \cos \theta\text{,}\) and \(p_n = 2p_1p_{n-1} - p_{n-2}.\) Then \(p_n = \cos(n\theta)\) for all integer \(n \geq 0\text{.}\)

A little trigonometric reminder

Recall that

\begin{equation*} \cos(a+b) = \cos a \cos b - \sin a \sin b \qquad \cos(a-b) = \cos a \cos b + \sin a \sin b \end{equation*}

Proof.

We prove the result by strong induction. When \(n=0\) we have \(p_0 = \cos 0 = 1\) as required. Now assume that \(p_j = \cos j \theta\) for \(j=0,1,2,\dots k\text{.}\) Now consider \(p_{k+1} = 2p_1 p_{k} - p_{k-1}\)

\begin{align*} p_{k+1} \amp = 2\cos \theta \cos k\theta - \cos(k-1)\theta\\ \amp = 2\cos\theta\cos k\theta - \left( \cos k\theta \cos \theta + \sin\theta\sin k\theta \right) \\ \amp = \cos\theta \cos k\theta - \sin\theta \sin k \theta \\ \amp = \cos((k+1)\theta) \end{align*}
as required. So the result holds for all integer \(n \geq 0\) by strong induction.
