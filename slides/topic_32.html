Topic 32—Images and preimages

Demirbaş & Rechnitzer

Images and preimages

Functions and subsets

How do functions interact with subsets of the domain and codomain?

Definition:

Let \(f:A \to B\) be a function and let \(C \subseteq A\) and \(D \subseteq B\)

  • The image of \(C\) in \(B\) is \(f(C) = \set{f(x) \text{ s.t. } x \in C}\)
  • The preimage of \(D\) in \(A\) is \(f^{-1}(D) = \set{x \in A \text{ s.t. } f(x)\in D}\)

WARNING — Be careful with preimages:

  • \(f^{-1}(x)\) is not \(\left(f(x)\right)^{-1}\) or \(\frac{1}{f(x)}\)
  • The preimage \(f^{-1}\) is not the inverse function.
  • When extra conditions satisfied the inverse function exists and we use the same notation
When you see \(f^{-1}\) think “preimage” — when you know inverse exists then “inverse function”.

A sketch of images and preimages

An example — images

Let \(f: \mathbb{R} \to \mathbb{R}\) be defined by \(f(x)=x^2\text{.}\) Then

  • \(\displaystyle f([0,4]) = [0,16]\)
  • \(\displaystyle f([-3,-1] \cup [1,2]) = [1,9]\)

  • If \(0 \leq x \leq 4\) then \(0 \leq x^2 \leq 16\)

  • If \(1 \leq x \leq 2\) then \(1 \leq x^2 \leq 4\text{.}\) And if \(-3 \leq x \leq -1\) then \(1 \leq x^2 \leq 9\text{.}\)

    So if \(x \in [-3,-1] \cup [1,2]\) then \(x^2 \in [1,9]\text{.}\)

An example — preimages

Let \(f: \mathbb{R} \to \mathbb{R}\) be defined by \(f(x)=x^2\text{.}\) Then

  • \(\displaystyle f^{-1}(\set{0,1}) = \set{-1,0,1}\)
  • \(\displaystyle f^{-1}([1,4]) = [-2,-1] \cup [1,2]\)

  • If \(x^2=0\) then \(x=0\text{.}\) And if \(x^2=1\) then \(x=\pm 1\)

    So if \(x^2 \in \set{0,1}\) then \(x \in \set{-1,0,1}\)

  • If \(1 \leq x^2\) then \(x \leq -1\) or \(x \geq 1\text{.}\) If \(x^2 \leq 4\) then \(-2 \leq x \leq 2\text{.}\)

    So if \(x^2 \in [1,4]\) then \(x \in [-2,-1]\) or \(x \in [1,2]\text{.}\)

Images, preimages and set operations

Images and preimages interact (mostly) nicely with subset, intersection and union.

Theorem:

Let \(f:A \to B\) and \(C \subseteq A\) and \(D \subseteq B\text{.}\) Then

\begin{equation*} C \subseteq f^{-1}(f(C)) \qquad \text{ and } \qquad f(f^{-1}(D)) \subseteq D \end{equation*}
Now let \(C_1,C_2 \subseteq A\) and \(D_1,D_2 \subseteq B\text{.}\) Then
\begin{align*} f(C_1 \cap C_2) \amp \subseteq f(C_1) \cap f(C_2) \amp f(C_1 \cup C_2) \amp = f(C_1) \cup f(C_2)\\ f^{-1}(D_1 \cap D_2) \amp = f^{-1}(D_1) \cap f^{-1}(D_2) \amp f^{-1}(D_1 \cup D_2) \amp = f^{-1}(D_1) \cup f^{-1}(D_2) \end{align*}

Make good problems — test lots of skills

Proof 1

\(f^{-1}(D_1 \cup D_2) = f^{-1}(D_1) \cup f^{-1}(D_2)\)

We use \(x \in f^{-1}(D) \iff f(x) \in D\)

Proof.

\(LHS \subseteq RHS\text{:}\) Let \(x \in f^{-1}(D_1 \cup D_2)\text{,}\) so \(f(x) \in D_1 \cup D_2\text{.}\)

Hence \(f(x) \in D_1\) or \(f(x) \in D_2\text{.}\)

  • when \(f(x)\in D_1\) we know \(x \in f^{-1}(D_1)\)
  • when \(f(x)\in D_2\) we know \(x \in f^{-1}(D_2)\)
In either case we know that \(x \in f^{-1}(D_1) \cup f^{-1}(D_2)\text{.}\)

Other inclusion is similar.

Proof 2

\(f(C_1 \cap C_2) \subseteq f(C_1) \cap f(C_2)\)

We use

\begin{align*} x \in C \amp \implies f(x) \in f(C) \amp \text{and}\amp\amp y \in f(C) \amp \implies \exists x \in C \text{ s.t. } y=f(x) \in f(C) \end{align*}

Proof.

Let \(y \in f(C_1 \cap C_2)\text{.}\)

This means that there is some \(x \in C_1\cap C_2\) so that \(f(x)=y\text{.}\) Then

  • since \(x \in C_1\text{,}\) we know that \(y=f(x)\in f(C_1)\)
  • since \(x \in C_2\text{,}\) we know that \(y=f(x)\in f(C_2)\)
Hence \(y \in f(C_1) \cap f(C_2)\text{.}\)

Reverse inclusion is false

\(f(C_1) \cap f(C_2) \not\subseteq f(C_1 \cap C_2)\)

Consider \(f: \mathbb{R} \to \mathbb{R}\) defined by \(f(x)=x^2\text{.}\)

  • Let \(C_1 = \set{-1}\text{,}\) so \(f(C_1) = \set{1}\)
  • Let \(C_2 = \set{1}\text{,}\) so \(f(C_2) = \set{1}\)
  • Then \(f(C_1) \cap f(C_2) = \set{1}\) but \(f(C_1 \cap C_2) = f(\es) = \es\)

Notice also that this shows \(f(x)\in f(C)\) does not imply \(x \in C\)

  • Set \(x=-1\) and \(C=\set{1}\)
  • Then \(f(x)=1\in \set{1}=f(C)\) but \(x \not\in C\text{.}\)

These fail because there are \(x_1 \neq x_2\) so that \(f(x_1) = f(x_2)\text{.}\)

Proof 3

\(C \subseteq f^{-1}(f(C))\)

Proof.

Let \(x \in C\text{.}\)

  • Since \(x \in C\text{,}\) we know that \(f(x) \in f(C)\)
  • To make logic clearer we write \(D = f(C)\text{,}\) so that \(f(x) \in D\)
  • Since \(f(x) \in D\text{,}\) we know that \(x \in f^{-1}(D)\)
  • But since \(D=f(C)\) this means that \(x \in f^{-1}(f(C))\) as required.

Reverse inclusion is false. Let \(f: \mathbb{R} \to \mathbb{R}\) with \(f(x)=x^2\text{.}\)

  • Let \(C = \set{2}\text{.}\) Then \(f(C) = \set{4}\)
  • But \(f^{-1}(\set{4}) = \set{-2,2}\) — since \(f(2)=f(-2)=4\text{.}\)
  • Thus \(f^{-1}(f(\set{2}))=\set{-2,2} \not\subseteq \set{2}=C\)
