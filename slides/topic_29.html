Topic 29—Set partitions

Demirbaş & Rechnitzer

Set partitions

Equivalence classes — equal or disjoint

Corollary:

Let \(R\) be an equivalence class on \(A\) and \(a,b \in A\text{.}\) Then

\begin{equation*} [a]=[b] \qquad \text{ or } \qquad [a] \cap [b] = \es \end{equation*}

Equal or disjoint

\([a]=[b]\) or \([a] \cap [b] = \es\)

Proof.

Let \(a,b \in A\) and consider the intersection \(C = [a] \cap [b]\text{.}\) Either \(C=\es\) or \(C \neq \es\text{.}\)

  • If \(C =\es\) we are done.

  • So now assume that \(C \neq \es\text{,}\) which means there is some \(c \in C\text{.}\) Hence \(c \in [a]\) and \(c \in [b]\text{.}\) Thus \(c \rel a\) and \(c \rel b\text{.}\)

    By symmetry we know \(a \rel c\text{,}\) and then transitivity gives \(a \rel b\text{.}\) The previous theorem then implies \([a]=[b]\) as needed.

Cutting up a set

Definition:

A partition of the set \(A\) is a set, \(\mathcal{P}\text{,}\) of non-empty subsets of \(A\) so that

  • if \(x \in A\) then there is \(X \in \mathcal{P}\) with \(x \in X\)
  • if \(X,Y \in \mathcal{P}\) then either \(X \cap Y = \es\) or \(X=Y\)
Elements of \(\mathcal{P}\) are parts or pieces of the partition.

Equivalence classes are a partition

Theorem:

Let \(R\) be an equivalence relation on \(A\text{.}\)

The set of equivalence classes of \(R\) forms a set partition.

Scratch work

  • Let \(\mathcal{P} = \set{[x] \st x \in A}\)

  • Need to show that every \(x \in A\) belongs to some \(X \in \mathcal{P}\)

    We already proved that each \(x\) belongs to \([x]\text{.}\)

  • Need to show that for each \(X,Y \in \mathcal{P}\text{,}\) either \(X \cap Y=\es\) or \(X=Y\)

    We just proved this!

Proof

Equivalence classes form a set partition

Proof.

Let \(\mathcal{P} = \set{[x] \st x \in A}\text{.}\)

  • Let \(x \in A\) then we proved previously that \(x \in [x]\text{.}\) Since \([x] \in \mathcal{P}\text{,}\) we know that \(x\) is in some piece of the partition.
  • Let \(X,Y \in \mathcal{P}\text{.}\) By the previous corollary we know that either \(X=Y\) or \(X \cap Y = \es\text{.}\)
Thus \(\mathcal{P}\) forms a set partition.

We can go further — a set partition can define an equivalence relation.

A set partition gives an equivalence relation

Theorem:

Let \(\mathcal{P}\) be a set partition of \(A\text{.}\) Now define a relation by

\begin{equation*} x \rel y \qquad \iff \qquad \exists X \in \mathcal{P} \text{ s.t. } x,y \in X \end{equation*}
then \(R\) is an equivalence relation.
