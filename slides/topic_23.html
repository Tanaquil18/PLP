Topic 23—Set operations

Demirbaş & Rechnitzer

Set operations

Union and intersection

Definition:

Let \(A, B\) be sets. The union of \(A\) and \(B\) is

\begin{align*} A \cup B &= \{x \st x\in A \text{ or } x \in B \} \end{align*}

Definition:

The intersection of sets \(A\) and \(B\text{,}\) is

\begin{align*} A \cap B &= \{ x \st x\in A \text{ and } x \in B \} \end{align*}
If the intersection \(A \cap B = \es\text{,}\) then we say that \(A\) and \(B\) are disjoint.

Notes and examples

  • Please use correct notation

    \begin{align*} A \cup B \amp= \text{ unions of sets} \amp A \cap B \amp= \text{ intersection of sets}\\ P \lor Q \amp = \text{ disjunction of statements} \amp P \land Q \amp= \text{ conjunction of statements} \end{align*}

  • There are parallels between set operations and logical operations but they are not the same

  • Let \(A = \set{1,2,3,4}, B = \set{p \st p \text{ is prime}}\) and \(C=\set{4,5,6,7}\)

    \begin{equation*} A \cup C = \set{1,2,3,4,5,6,7} \qquad A \cap B = \set{2,3} \qquad B \cap C = \set{5,7} \end{equation*}

Another operation

Definition:

Let \(A\) and \(B\) be sets. Then the difference, \(A - B\) is

\begin{align*} A - B &= \{ x \in A \st x \notin B \} \end{align*}

Universals and complements

Definition:

Given a universal set \(U\) and \(A \subset U\text{,}\) the complement of \(A\) is

\begin{align*} \bar{A} \amp = \{ x \in U \st x \notin A\} \amp \text{or equivalently} \amp \amp x \in \bar{A} \iff x \not\in A \end{align*}
The universal set is the set from which we draw elements in the current context

Notes and examples

  • Compl-e-ment vs Compl-i-ment
  • \(A - B\) also written \(A \setminus B\) and is called the relative complement of \(B\) in \(A\)
  • We have \(A - B = A \cap \bar{B}\)

  • Let \(U= \mathbb{N}, A = \set{1,2,3,4}, B = \set{p \st p \text{ is prime}}\) and \(C=\set{4,5,6,7}\)

    \begin{equation*} A - C = \set{1,2,3} \qquad A - B = \set{1,4} \qquad \bar{A} = \set{n \in \mathbb{N} \st n \geq 5} \end{equation*}

Ordered pairs

Sets don't care about order, but many applications need pairs of objects.

Definition:

An ordered pair of elements is an ordered list of two elements.

The ordered pair of two elements \(a,b\) is written \((a,b)\) and satisfies

  • \((a,b) = (c,d)\) only when \((a=c)\) and \((b=d)\text{,}\) and
  • \((a,b) \neq (b,a)\) unless \((a=b)\text{.}\)

Warning

  • use correct notation: \(\set{1,3}\) is a set, \((1,3)\) is an ordered pair
  • we sometimes use \((1,3) = \set{x \in \mathbb{R} \st 1 \lt x \lt 3}\) — give your reader context

Cartesian product

Definition: Cartesian product.

The Cartesian product of sets \(A,B\) is

\begin{equation*} A \times B = \set{ (a,b) \st a\in A, b\in B} \end{equation*}
Note for \(A,B\neq \es\text{,}\) \(A\times B \neq B \times A\) unless \(A=B\text{.}\)

Let \(A = \set{a,b,c}\) and \(B=\set{1,2}\) then

\begin{align*} A\times B &= \set{ (a,1), (a,2), (b,1), (b,2), (c,1), (c,2)} \end{align*}
