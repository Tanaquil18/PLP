Topic 43 — More infinities

Demirbaş & Rechnitzer

Cantor's theorem and more infinities

Comparing different infinities

We know that \(\mathbb{N} \subset \mathbb{R}\) and we proved that \(|\mathbb{N}| \neq |\mathbb{R}|\text{.}\) So want to state

\begin{equation*} \aleph_0 = |\mathbb{N}| \lt |\mathbb{R}|= c \end{equation*}
We can make this precise by extending ideas from finite sets \(A,B\text{:}\)
  • If \(f:A \to B\) is an injection then \(|A| \leq |B|\)
  • If \(h:A \to B\) is an bijection then \(|A| = |B|\)

Definition:

Let \(A,B\) be sets.

  • We write \(|A| \leq |B|\) when there is an injection from \(A\) to \(B\text{.}\)

  • Further, we write \(|A| \lt |B|\) when there is an injection from \(A\) to \(B\) but no bijection.

    \begin{equation*} |A| \lt |B| \qquad \iff \qquad \big(|A| \leq |B|\big) \land \big(|A| \neq |B|\big) \end{equation*}

Good exercise: if \(|A| \leq |B|\) prove there is a surjection from \(B\) to \(A\text{.}\)

Continuum hypothesis

  • Cantor's diagonal argument proves that

    \begin{equation*} \aleph_0 = |\mathbb{N}| \lt |\mathbb{R}| =c \end{equation*}

  • Is there any infinity between these two? More precisely?

    \begin{equation*} \exists A \text{ s.t. } | \mathbb{N} | \lt |A| \lt | \mathbb{R} | \end{equation*}

Conjecture 2. Continuum hypothesis (Cantor 1878).

There is no set \(A\) so that \(\aleph_0 \lt |A| \lt c\text{.}\)

  • Gödel (1940) showed that it cannot be disproved from standard set theory axioms (Zermelo–Fraenkel)
  • Cohen (1963) showed that it cannot be proved from standard set theorem axioms
  • So (technically) not really correct to call it a conjecture

Bigger infinities

Are there bigger infinities?

Theorem: (Cantor's theorem, 1891).

Let \(A\) be a set. Then \(|A| \lt |\pow{A}|\)

Scratch work

  • Easy to find an injection from \(A\) to \(\pow{A}\text{.}\) Here are two examples

    \begin{align*} f:\amp A \to \pow{A} \amp f(a) \amp = \set{a} \\ h:\amp A \to \pow{A} \amp h(a) \amp = A-\set{a} \end{align*}
    This proves that \(|A| \leq |\pow{A}|\)

  • We prove there is no bijection from \(A\) to \(\pow{A}\) by showing there cannot be a surjection

Good and bad

To explore, let \(A=\set{1,2,3}\) and consider \(f,h\) from previous slide.

\begin{align*} f(1) \amp= \set{1} \amp f(2) \amp = \set{2} \amp f(3) \amp = \set{3} \\ h(1) \amp= \set{2,3} \amp h(2) \amp = \set{1,3} \amp h(3) \amp = \set{1,2} \end{align*}
Notice that
  • \(\displaystyle \forall x \in A, x \in f(x)\)
  • \(\displaystyle \forall x \in A, x \not\in h(x)\)

More generally, if we have any function \(g:A \to \pow{A}\) then

  • if \(x \in g(x)\) then call \(x\) a good point, and
  • if \(x \not\in g(x)\) then call \(x\) a bad point
Then build sets of all the good and bad points
\begin{equation*} G = \set{x \in A \text{ s.t. } x \in g(x)} \qquad \text{and} \qquad B = \set{x \in A \text{ s.t. } x \not\in g(x)} \end{equation*}
Notice that \(G,B \subseteq A\) and so \(G,B \in \pow{A}\text{.}\)

The bad set is more interesting

Proof.

Assume, to the contrary that there is a surjection \(g: A \to \pow{A}\)

  • Construct the “bad” set \(B = \set{x \in A \text{ s.t. } x \not\in g(x)} \subseteq A\)
  • Now since \(B \in \pow{A}\) and \(g\) is surjective, there must be some \(b\in A\) so that \(g(b)=B\)

  • We must have that either \(b \in B\) or \(b \not\in B\text{?}\) Is it good or bad?

    • When \(b \in B\text{,}\) by definition of \(B\) must have \(b \not\in B\) — contradiction
    • When \(b \not\in B\text{,}\) by definition of \(B\) must have \(b \in B\) — contradiction

  • These contradictions mean there is no \(b\) so that \(g(b)=B\text{,}\) and so \(g\) is not surjective
Then since we have constructed an injection from \(f:A \to \pow{A}\text{,}\) it follows that \(|A| \lt |\pow{A}|\text{.}\)

This immediately gives \(| \mathbb{N} | \lt | \pow{\mathbb{N}}|\)

With work you can prove that \(| \pow{\mathbb{N}}| = |\mathbb{R}|\) — see Cantor-Schröder-Bernstein Theorem

Keep going

\(|A| \lt |\pow{A}|\) and \(|\mathbb{N}| \lt |\pow{\mathbb{N}}|\)

Do it again — \(|\mathbb{N}| \lt |\pow{\mathbb{N}}| \lt |\pow{\pow{\mathbb{N}}}|\)

And again — \(|\mathbb{N}| \lt |\pow{\mathbb{N}}| \lt |\pow{\pow{\mathbb{N}}}| \lt |\pow{\pow{\pow{\mathbb{N}}}}|\)

Corollary:

There are an infinite number of different infinites.

Proof.

Starting with \(\mathbb{N}\text{,}\) Cantor's theorem tells us that \(\pow{\mathbb{N}}\) is a larger infinite set. Then \(\pow{\pow{\mathbb{N}}}\) is larger again. By repeatedly taking power sets, you create an infinitely long sequence of larger and larger infinite sets.
