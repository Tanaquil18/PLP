Topic 34—Compositions

Demirbaş & Rechnitzer

Compositions

Chaining functions together

Definition:

Let \(f: A \to B\) and \(g: B \to C\text{.}\)

The composition of \(f\) and \(g\text{,}\) denoted \(g \circ f\text{,}\) defines a new function

\begin{align*} g\circ f: A \to C && \left(g\circ f\right)(a) = g\left(f(a)\right) \qquad\forall a \in A \end{align*}
Note composition is associative: \(h \circ (g \circ f) = (h \circ g) \circ f\text{.}\)

Be careful of notation — read left-to-right but apply right-to-left: \((h\circ g \circ f)(x) = h(g(f(x)))\)

Compositions, injections and surjections

Compositions play nicely with injections and surjections.

Theorem:

Let \(f:A \to B\) and \(g: B \to C\) be functions.

  • If \(f\) and \(g\) are injective then so is \(g\circ f\text{.}\)
  • If \(f\) and \(g\) are surjective then so is \(g\circ f\text{.}\)
Consequently if \(f,g\) are bijective then so is \(g \circ f\text{.}\)

Composition of injections

Use injection property — different map to different

Proof.

Let \(a_1,a_2 \in A\) so that \(a_1 \neq a_2\text{.}\)

Since \(f\) is injective, we know that \(f(a_1) \neq f(a_2)\text{.}\) And thus since \(g\) is injective, we know that \(g(f(a_1)) \neq g(f(a_2))\text{.}\)

Thus \((g\circ f)(a_1) \neq (g \circ f)(a_2)\) as required.

Composition of surjections

Use surjection property — everything is mapped to by something

Proof.

Let \(c \in C\text{.}\)

Since \(g\) is surjective, we know that there is \(b \in B\) so that \(g(b)=c\text{.}\) Then since \(f\) is surjective, we have some \(a \in A\) so that \(f(a)=b\text{.}\)

Thus \(g(f(a))=g(b)=c\text{,}\) and so for any \(c \in C\) we can find \(a \in A\) so that \((g\circ f)(a) =c\) as required.

Partial converse

Theorem:

Let \(f:A \to B\) and \(g:B \to C\) be functions, then

  • if \(g \circ f\) is an injection then \(f\) is an injection.
  • if \(g \circ f\) is a surjection then \(g\) is a surjection.

The proofs of these statements make excellent exercises.

Note that you cannot extend this to a full converse. There exist \(f,g\) so that

  • \(g \circ f\) is an injection, but \(g\) is not injective
  • \(g \circ f\) is a surjection, but \(f\) is not surjective

Disproof of full converse

  • \(g \circ f\) is an injection, but \(g\) is not injective
  • \(g \circ f\) is a surjection, but \(f\) is not surjective

  • Since \(g(f(1)) \neq g(f(2))\text{,}\) \(g \circ f\) is injective. But \(g(4)=g(5)\text{,}\) so \(g\) not an injection.
