Topic 33—Injections, surjections and bijections

Demirbaş & Rechnitzer

Injections, surjections and bijections

Injections — different maps to different

Definition:

Let \(f:A \to B\) be a function.

The function \(f\) is injective when for all \(a_1,a_2 \in A\)

\begin{equation*} (a_1 \neq a_2) \implies f(a_1)\neq f(a_2) \end{equation*}
Equivalently (by the contrapositive)
\begin{equation*} f(a_1) = f(a_2) \implies a_1 = a_2 \end{equation*}
Injections are also called one-to-one functions.

Note: if \(f\) is injective then for every \(b \in B, |f^{-1}(\set{b})| \leq 1\text{.}\)

This is consistent with \(|A| \leq |B|\)

Injection example

Proposition:

The function \(f: \mathbb{R} \to \mathbb{R}\) defined by \(f(x)=7x-3\) is injective

Use \(f(a_1)=f(a_2) \implies a_1=a_2\) to prove — equalities easier than inequalities.

Proof.

Let \(x,z \in \mathbb{R}\) and assume that \(f(x)=f(z)\text{.}\) Then we know that

\begin{align*} 7x-3 \amp= 7z-3 \\ 7x\amp=7z \\ x\amp=z \end{align*}
and hence the function is injective.

Surjections — everything is mapped to by something

Definition:

Let \(g:A \to B\) be a function.

The function \(g\) is surjective when

\begin{equation*} \forall b \in B, \exists a \in A \text{ s.t. } g(a)=b \end{equation*}
Surjections are also called onto functions.

Note: if \(g\) is surjective then for every \(b \in B, |g^{-1}(\set{b})| \geq 1\text{.}\)

This is consistent with \(|A| \geq |B|\)

Surjection example

Proposition:

The function \(f: \mathbb{R} \to \mathbb{R}\) defined by \(f(x)=7x-3\) is surjective

Given \(y \in \mathbb{R}\text{,}\) we need to find \(x\in \mathbb{R}\) so that \(f(x)=y\text{:}\)

\begin{equation*} y=7x-3 \qquad \text{so} \qquad y+3=7x \qquad \text{so} \qquad x = \frac{y+3}{7}. \end{equation*}

Proof.

Let \(y \in \mathbb{R}\) and set \(x = \frac{y+3}{7} \in \mathbb{R}\text{.}\) Then

\begin{equation*} f(x) = 7x-3 = 7\cdot \frac{y+3}{7}-3 = y+3 -3 = y \end{equation*}
as required. Hence the function is surjective.

A non-example

Proposition:

The function \(f: \mathbb{R} \to \mathbb{R}\) defined by \(f(x)=x^2\) is neither injective nor surjective.

\begin{align*} \text{not injective } \amp \equiv \exists x_1,x_2 \in A \text{ s.t. } (x_1\neq x_2) \land (f(x_1)=f(x_2))\\ \text{not surjective} \amp \equiv \exists b \in B \text{ s.t. } \forall a \in A, f(a) \neq b \end{align*}

Proof.

We prove each claim in turn.

  • Now let \(x=1,z=-1\text{.}\) Then since \(f(x)= 1 = f(z)\text{,}\) the function is not an injection.

  • Let \(y = -1\text{.}\) For any \(x \in \mathbb{R}\) we know \(f(x)=x^2 \geq 0\text{,}\) so there is no \(x \in \mathbb{R}\) so that \(f(x)=y\text{.}\) So \(f\) is not a surjection.

Bijections — injective and surjective

Definition:

Let \(h:A \to B\) be a function. The function \(h\) is bijective when it is both injective and surjective.

Bijections are also called one-to-one correspondences.

Note: if \(h\) is bijective then

  • since \(h\) is injective we know that for every \(b \in B, |h^{-1}(\set{b})| \leq 1\)
  • since \(h\) is surjective we know that for every \(b \in B, |h^{-1}(\set{b})| \geq 1\)
and so for every \(b \in B, |h^{-1}(\set{b})|=1\text{.}\)

This is consistent with \(|A| = |B|\)

From work above \(f: \mathbb{R} \to \mathbb{R}\) defined by \(f(x)=7x-3\) is bijective.

Another \(x^2\) example

Consider the 4 functions

\begin{align*} f :\amp\mathbb{R} \to \mathbb{R} \amp f(x) = x^2\\ g :\amp\mathbb{R} \to [0,\infty) \amp g(x) = x^2\\ h :\amp [0,\infty) \to \mathbb{R} \amp g(x) = x^2\\ \rho :\amp [0,\infty) \to [0,\infty) \amp \rho(x) = x^2 \end{align*}

Then

  • \(f\) is neither injective nor surjective
  • \(g\) is surjective but not injective
  • \(h\) is injective but not surjective
  • \(\rho\) is both injective and surjective
Let's prove the last one carefully.

Injective and surjective

\(\rho : [0,\infty) \to [0,\infty)\) with \(\rho(x) = x^2\) is injective and surjective

Proof.

We prove each in turn.

  • Injection: Let \(x,z \in [0,\infty)\) with \(\rho(x)=\rho(z)\text{.}\) Then we know that \(x^2=z^2\) and hence \(x = \pm z\text{.}\) But since \(x,z \geq 0\) we must have \(x=z\) as required.
  • Surjection: Let \(y \geq 0\) and then set \(x = \sqrt{y}\text{.}\) Since \(y \geq 0\text{,}\) we know that \(x \in \mathbb{R}\text{.}\) Then \(\rho(x) = x^2 = (\sqrt{y})^2 = y\text{.}\)
