Topic 18—Induction

Demirbaş & Rechnitzer

Induction

Induction

Mathematical induction is a specialised technique for proving

\begin{equation*} \forall n \in \mathbb{N}, P(n) \end{equation*}

It breaks the proof into 2 simpler steps

base case

prove that \(P(1)\) is true

inductive step

show that \(P(k) \implies P(k+1)\)

  • base case \(=\) step onto bottom rung of ladder
  • inductive step \(=\) from the current rung you can reach the next rung
  • so you can climb the ladder as high as you want

An example

Proposition:

For all \(n \in \mathbb{N}\text{,}\) \(n^2+5n-7\) is odd

scratch work

  • base case when \(n=1\) we have \(1+5-7 = -1\) which is odd

  • inductive step need to prove

    \begin{equation*} (k^2+5k-7 \text{ is odd}) \implies ( (k+1)^2+5(k+1)-7 \text{ is odd}) \end{equation*}

  • The inductive step is a sub-proof inside our proof

Inductive step “sub proof”

\(\forall k \in \mathbb{N}, (k^2+5k-7 \text{ is odd}) \implies ( (k+1)^2+5(k+1)-7 \text{ is odd})\)

scratch work

  • so we assume \(k^2+5k-7 = 2\ell+1\)
  • need to show \((k^2+2k+1)+5(k+1)-7 = \underbrace{(k^2+5k-7)}_{2\ell+1}+(2k+6)\) is odd.

  • Since \(k^2+5k-7 = 2\ell+1\) we know

    \begin{equation*} (k+1)^2+5(k+1)-7 = 2(\ell+k+3)+1 \end{equation*}
    and since \(\ell+k+3 \in \mathbb{Z}\) we are done.

Of course we still need to put the two parts of the proof together.

Principle of mathematical induction

Theorem: Mathematical induction.

For all \(n\in\mathbb{N}\) let \(P(n)\) be a statement. Then if

  • \(P(1)\) is true, and
  • \(P(k) \implies P(k+1)\) is true for all \(k \in \mathbb{N}\)
then \(P(n)\) is true for all \(n \in \mathbb{N}\text{.}\)

Warnings

  • Induction is not “adding the next term to both sides”
  • Induction does not prove all statements — the law of the instrument
  • Tell your reader if you use induction in your proof

Completing our proof

For all \(n \in \mathbb{N}\text{,}\) \(n^2+5n-7\) is odd

Proof.

We prove the result by induction.

  • Base case: When \(n=1\) we have \(1+5-7=-1\) which is odd.

  • Inductive step: Assume that \(k^2+5k-7\) is odd, so we can write

    \begin{align*} k^2+5k-7 \amp = 2\ell+1 \text{ for some } \ell \in \mathbb{Z} \text{ and so}\\ (k+1)^2+5(k+1)-7 \amp= 2(\ell+k+3)+1 \end{align*}
    and since \(\ell+k+3 \in \mathbb{Z}\text{,}\) it follows that \((k+1)^2+5(k+1)-7\) is odd.

Since the base case and inductive step hold, the result follows by induction.

Another example

Proposition:

For every natural number \(n\text{,}\) \(3 \mid (4^n-1)\)

Scratch work

  • When \(n=1\text{,}\) easy \(3 \mid (4-1)\)
  • Assume \(3 \mid (4^k-1)\text{,}\) so \(4^k-1 = 3\ell\text{.}\)

  • Writing \(4^k = 3\ell+1\) shows

    \begin{equation*} 4^{k+1} = 12\ell+4 \qquad \text{ so }\qquad 4^{k+1}-1 = 12 \ell +3 \end{equation*}
    done!

Write it up nicely

For every natural number \(n\text{,}\) \(3 \mid (4^n-1)\)

Proof.

We prove the result by induction.

  • Base case: When \(n=1\) we have \(3\mid(4-1)\text{,}\) so the result holds.

  • Inductive step: Assume that \(3 \mid (4^k-1)\text{,}\) so \(4^k = 3 \ell+1\) for some \(\ell \in \mathbb{Z}\text{.}\) Then

    \begin{equation*} 4^{k+1}-1 = 4(3\ell+1)-1 = 3(4\ell +1) \end{equation*}
    and so \(3 \mid (4^{k+1}-1)\) as required.

Since the base case and inductive step hold, the result follows by induction.
